# GR-calculation
## はじめに
本リポジトリは、GR(General Relativity)における添え字計算をどうにか自動化できないかと思い、試しに作ってみているものである。物理学のエキスパートから見てもプログラミングのエキスパートから見ても、まだまだなところはあるが随時改善し更新していく予定である。

- 使用言語：python<br>
- 使用パッケージ：<br>
  - sympy<br>
  - einsteinpy.symbolic

## 計算内容
まだ具体的に計算したわけではないが、現在のコードは<br>
- シュワルツシルト計量を定義
- 定義したシュワルツシルト計量の成分を行列として表示<br>

までを行うものになっている。なお、符号はalmost plusとする。

$$
\begin{pmatrix}
-1+\frac{r_{s}}{r}&0&0&0\\
0&\frac{1}{1-\frac{r_{s}}{r}}&0&0\\
0&0&r^{2}&0\\
0&0&0&r^{2}{\sin}^{2}{\theta}
\end{pmatrix}
$$
